AvP Uzay Simülasyonu

Bu proje, Algoritma ve Programlama dersi kapsamında bireysel olarak geliştirilmiştir.
Program, farklı gezegenlerin yerçekimi ivmelerini kullanarak çeşitli fizik deneylerini C dili ile simüle etmektedir.

Proje Amacı

Bu çalışmanın amacı;

C programlama dilinde fonksiyonel ve modüler yapı kurabilmek,

Dizi ve pointer kullanımını pekiştirmek,

Fiziksel formülleri algoritmik yapıya dökebilmek,

Kullanıcıdan alınan girdilerle gezegenlere göre karşılaştırmalı sonuçlar üretebilmektir.

Programın Genel Çalışma Akışı

Kullanıcıdan bilim insanının adı alınır

Menü üzerinden deney seçimi yapılır

Deneye ait gerekli fiziksel büyüklükler kullanıcıdan alınır

Seçilen deney, tüm gezegenler için ayrı ayrı hesaplanır

Sonuçlar konsol ekranına yazdırılır

İçerik

C kaynak kodu: main.c

Proje raporu: rapor/ klasörü

Ekran görüntüleri: ekran_goruntuleri/ klasörü
